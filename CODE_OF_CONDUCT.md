# 📋 Code of Conduct - Gaianexus

## Notre engagement

Gaianexus s'engage à fournir un environnement accueillant et inclusif pour tous les participants, indépendamment de leur :
- Âge, apparence physique, handicap
- Origine ethnique, identité ou expression de genre
- Niveau d'expérience, nationalité
- Opinions politiques ou religieuses
- Orientation sexuelle

Nous nous engageons à créer une communauté libre de harcèlement et de discrimination.

## Nos standards

### ✅ Comportements attendus

- Utilisez un langage accueillant et inclusif
- Soyez respectueux des opinions différentes
- Acceptez les critiques constructives avec grâce
- Concentrez-vous sur ce qui est le mieux pour la communauté
- Montrez de l'empathie envers les autres participants
- Favorisez la collaboration plutôt que la compétition

### ❌ Comportements inacceptables

Ceux-ci incluent, sans s'y limiter :
- Langage ou imagerie sexuelle
- Attaques personnelles ou insultes
- Harcèlement public ou privé
- Divulgation d'informations privées sans consentement
- Conduite considérée comme inappropriée dans un cadre professionnel
- Discrimination ou exclusion basée sur les caractéristiques énumérées ci-dessus
- Désinformation intentionnelle
- Comportements abusifs ou intimidants

## Faire respecter ce code

### Signaler un problème

Si vous êtes témoin ou victime d'un comportement inacceptable :

1. **Documentez** l'incident (date, heure, description)
2. **Contactez** les mainteneurs via email : `vinztao@example.com`
3. **Signalez** via GitHub (option "Report abuse")

**Confidentiel** : Tous les rapports seront traités de manière confidentielle.

### Réaction aux signalements

Les mainteneurs vont :
1. Examiner le rapport
2. Mener une enquête si nécessaire
3. Prendre les mesures appropriées

Les mesures peuvent inclure :
- Un avertissement privé
- Une demande de modification du comportement
- Une suspension temporaire
- Une suppression permanente du projet

## Portée

Ce code de conduite s'applique à :
- Tous les espaces liés à Gaianexus (GitHub, forums, réunions, événements)
- Les interactions officielles ou représentant le projet
- Les communications de projet
